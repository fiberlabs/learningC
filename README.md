# Learning C

This is my personal repository for learning C programming,
starting from zero and building up fundamentals.

## Structure

- `01-setup/` – Environment setup, Hello World
- `02-basics/` – Variables, I/O, types, arithmetic

More modules will be added as I progress.

## Build & Run

Each `.c` file is self-contained. To compile and run:

```bash
gcc -Wall -Wextra file.c -o file
./file
```
