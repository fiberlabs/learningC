# Module 2 – C Basics

Variables, data types, input/output, arithmetic, and building small programs.

## Projects

- `project02-madlibs_game` – A word-substitution story game using `scanf` and `printf`
- `project03-age_calculator` – Calculates age and heartbeats, avoiding integer overflow
- `project04-server_planner` – Data center invoice with large number handling
- `project05-data_inspector` – Inspects characters and integers in ASCII, hex, and memory

## Covered

- Variables (`int`, `double`, `char`, arrays)
- `printf` with format specifiers, escape sequences
- `scanf` and the address-of operator `&`
- Integer division, modulo, type casting
- `sizeof`, fixed-width types (`stdint.h`), portable printing (`inttypes.h`)
- Compiler warnings (`-Wall -Wextra`)
