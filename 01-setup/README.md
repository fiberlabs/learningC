# Module 1 – Environment Setup

Setting up the Linux terminal, GCC compiler, VS Code, Git, and compiling my first C program.

## Project

- `project01-hello_world` – Prints a confirmation message and proves the toolchain works.

## Covered

- Terminal navigation
- Installing `build-essential` and Git
- VS Code extensions for C
- SSH keys and GitHub connection
- `.gitignore` for ignoring binaries and practice files
- Compilation with `gcc` and running the executable
