# Module 4 – String Manipulation (Arrays & Pointers)

Working with C strings, raw memory, character analysis, and safe text handling.

## Projects

- `project19-memory_buffer_playground` – Raw memory operations using `memcpy`, `memmove`, `memset`, and `memcmp`
- `project20-scrolling_text_effect` – Pointer arithmetic to create a scrolling text effect
- `project21-input_sanitizer` – Detects dangerous shell characters using `ctype.h`
- `project22-RPG_status_screen_generator` – Generates a formatted UI using `snprintf`
- `project23-username_and_email_formatter` – Trims whitespace, normalizes input, and builds an email address
- `project24-password_strength_checker` – Scores passwords using character analysis
- `project25-text_analyzer` – Counts characters, words, and sentences using a finite state machine

## Covered

- Character arrays and the null terminator (`'\0'`)
- `strlen` vs `sizeof`
- String comparison with `strcmp`
- Copying and concatenation (`strcpy`, `strcat`)
- Safe input with `fgets` and newline removal using `strcspn`
- Raw memory functions (`memcpy`, `memmove`, `memset`, `memcmp`)
- Pointer arithmetic and string slicing
- String searching with `strchr` and `strstr`
- Character analysis (`isupper`, `islower`, `isdigit`, `isspace`, `ispunct`, `iscntrl`)
- Character conversion (`toupper`, `tolower`)
- Safe formatting with `snprintf`
